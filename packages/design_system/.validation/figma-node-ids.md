## States Coverage
Each button variant has the following states:
- default
- hover
- focus
- disabled
- pressed
- loading

## Icon Configurations
Each button can have:
- No icon (Label=on, Icon=off)
- Left icon (Label=on, Icon=on, IconPos=left)
- Right icon (Label=on, Icon=on, IconPos=right)
- Icon only (Label=off, Icon=on)
